= Migration from admin-faces
:toc:

This project is inspired and based on https://adminfaces.github.io/site/[AdminFaces] by https://github.com/rmpestano[Rafael Pestano].

This document describes how to upgrade a SpringBoot based AdminFaces project.

== Dependencies

Remove any all AdminFaces based dependencies and replace them with the pf-adm dependency.

[cols="a,a"]
|===
^| AdminFaces ^| PF-Adm

|
[source,xml]
----
<dependency>
  <groupId>com.github.adminfaces</groupId>
  <artifactId>admin-template</artifactId>
</dependency>
<dependency>
  <groupId>com.github.adminfaces</groupId>
  <artifactId>admin-theme</artifactId>
</dependency>
<dependency>
  <groupId>com.primefaces</groupId>
  <artifactId>primefaces</artifactId>
</dependency>
----

if based on Joinfaces

[source,xml]
----
<dependency>
  <groupId>org.joinfaces</groupId>
  <artifactId>adminfaces-spring-boot-starter</artifactId>
</dependency>
----

|
[source,xml]
----
<dependency>
  <groupId>de.muehlencord.pf-adm</groupId>
  <artifactId>pf-adm-spring-boot-starter</artifactId>
</dependency>
----
|===
